나만 알고 있는 우리 동네 미지의 공간을 특별한 큐레이팅을 통해 소개하는 서비스

## ⛓ LINK

[미(간)지 사이트 바로가기](https://miganzi.vercel.app/)

## 실행 방법

1. 깃 클론

```
git clone https://github.com/mi-gan-zi/FE-MiGanZi.git
```

2. 패키지 인스톨

```
yarn install
```

3. 로컬환경 테스트 실행

```
yarn start
```
